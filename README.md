# 🤖 Rule-Based AI Chatbot

A simple Rule-Based AI Chatbot built using Python and Streamlit. The chatbot responds to predefined user queries using rule-based logic and provides an interactive web interface.

## 🌐 Live Demo

https://manaswini-ai-chatbot.streamlit.app

## 📂 GitHub Repository

https://github.com/manaswiniduvva-hub/AI-CHATBOT

## ✨ Features

* Interactive chat interface
* Multiple predefined responses
* AI and programming related questions
* General knowledge questions
* Chat history support
* Exit commands (exit, quit, close)
* Streamlit web deployment

## 🛠️ Technologies Used

* Python
* Streamlit
* Git
* GitHub

## 🚀 How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/manaswiniduvva-hub/AI-CHATBOT.git
```

2. Navigate to the project folder

```bash
cd AI-CHATBOT
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
streamlit run app.py
```
